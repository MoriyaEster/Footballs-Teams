# Instructions to Set Up and Run the Project

## Prerequisites:
1. Install **Node.js**: Download it from https://nodejs.org/.
2. Verify installation:
   - Run `node -v` to ensure Node.js is installed.
   - Run `npm -v` to verify npm is installed.

---

## Steps to Set Up:

1. Clone the repository: git clone https://github.com/MoriyaEster/Footballs-Teams.git

2. Install all dependencies:
    npm install
    axios react-router-dom lodash
